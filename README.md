# AI Lecture Voice-to-Notes Generator

An AI-powered application that converts lecture audio into structured study materials such as notes, quizzes, flashcards, and MCQs using speech-to-text and generative AI.

## Features
- Lecture audio transcription using Whisper
- AI-generated study notes
- Quiz and MCQ generation
- Flashcards for revision
- Lecture-based chatbot
- Streamlit web interface

## Technologies Used
- Python
- Streamlit
- Whisper (Speech-to-Text)
- FLAN-T5 (Generative AI)
- Hugging Face Transformers
- PyTorch

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
